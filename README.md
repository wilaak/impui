# <img src="assets/impui.svg" alt="imp" width="320">

> "Give someone state and they'll have a bug one day, but teach them how to represent state in two separate locations that have to be kept in sync and they'll have bugs for a lifetime." -[ryg](https://twitter.com/rygorous/status/1507178315886444544)

Delightfully simple immediate-mode flavored server rendered UI's in PHP. Take HATEOAS to the next level with IMPUI!

## Features

Powered by the terrific [Datastar](https://data-star.dev/): *The Hypermedia Framework*.

- Realtime SSE or simple request/response.
- UI = fn(state): Easy to reason about, easy to write, easy to test!
